### 2018-1-C: 数组转换

**总时间限制**: 1000ms **内存限制**: 65535kB

**描述**

有$n$个数字$1,2,3…,n$，由小至大依次放入栈。问是否存在一种出栈方式，使这$n$个数字出栈的顺序为$a1,a2,…,an$。

**输入**

第一行输入一个整数t，代表有t组测试数据 对于每组测试数据，第一行输入整数n，第二行输入n个数字：$a1,a2, … , an 1<= n <= 1000$

**输出**

每组测试数据输出一行。 如果出栈顺序$a1,a2,…,an$是合法的，输出`yes`，否则输出`no`。

**样例输入**

```
3
3
1 2 3
3
3 2 1
4
1 4 2 3
```

**样例输出**

```
yes
yes
no
```
